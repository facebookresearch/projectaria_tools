# Project Aria Glasses Cable Clip

## Overview

A glasses cable clip is available to hold the Project Aria glasses cable onto the glasses. This can be useful if connecting glasses to a battery pack while recording.

If you need this accessory, please contact [AriaOps@meta.com](mailto:AriaOps@meta.com).


## How to use

1. Unlatch the clip
2. Slide the clip onto the Aria glassesâ€™ arm
3. Place the charger onto the magnetic port
4. Close the clip using the rubber band
