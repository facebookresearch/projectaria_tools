How to Use CMake with Project Aria Tools

To use projectaria_tools for your own projects with CMake, we recommend adding it as a submodule in your project.

  1. Add the submodule
cd /path_to_your_project

git submodule add https://github.com/facebookresearch/projectaria_tools.git
  1. In your project root CMakeLists.txt, add the projectaria_tools subdirectory
add_subdirectory(projectaria_tools)
  1. Link your project targets (libraries or executables) to projectaria_tools modules. For example, to link with the sensor_calibration library, use:
target_link_libraries(my_target
  PUBLIC
    sensor_calibration
)
