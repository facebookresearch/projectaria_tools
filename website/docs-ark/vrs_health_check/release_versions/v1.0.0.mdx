# VRS Health Check 1.0.0 Release Notes

## Overview

This is the initial release of VRS Health Check as a standalone Python package.

### Core Functionality
- **Comprehensive VRS file validation**: Check the integrity and quality of VRS recordings
- **Multi-sensor health checks**: Validate data from cameras, IMUs, audio, and other sensors
- **Configurable thresholds**: Customize validation criteria for your specific needs
- **Detailed reporting**: Get comprehensive reports on file health status
