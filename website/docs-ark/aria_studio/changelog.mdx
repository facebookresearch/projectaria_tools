# Changelog

---
## Version 2.0.0
*June 2026*

Aria Studio 2.0.0 adds support for Project Aria Gen 2 glasses:

- **Device Discovery & Control:** Automatic USB detection with certificate-based pairing via the Companion App, multi-device support with one-click switching
- **Live Streaming:** Stream sensor data in real time over USB into a Rerun 3D viewer
- **Custom Recording:** Configure and start recordings from the desktop with selectable profiles for multi-device sync
- **MPS Integration:** Request SLAM and Hand Tracking directly from the UI, download results automatically and visualize in Rerun
- **Groups & Multi-SLAM:** Organize recordings into groups and submit Multi-SLAM requests that produce trajectories in a shared coordinate frame
- **Full Documentation Site:** 12 new pages covering every Aria Studio workflow, from getting started to troubleshooting
