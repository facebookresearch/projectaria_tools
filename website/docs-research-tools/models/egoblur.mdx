# EgoBlur

## Overview
EgoBlur models are open source AI models from Meta to preserve privacy by detecting and blurring faces and license plates from images. We provide two FasterRCNN-based detector models.

For recordings captured by Aria Gen2 devices, we provide both face and license plate anonymization models.
* Face model is fine-tuned on the [EgoBlur Gen1 face model](https://facebookresearch.github.io/projectaria_tools/docs/open_models/egoblur) with 250k images, 316k bounding boxes
  * 90k images with 292k bounding boxes from non-egocentric data. 160k images with 24k bounding boxes from Aria Gen2 data.
* License plate model is trained on 622k images, 676k bounding boxes
  * 584k images with 629k bounding boxes from non-egocentric data. 38k images with 47k bounding boxes from Aria Gen1 data.
* The EgoBlur face and license plate models are approximately 400 MB each and have ~104 million parameters.
* The models are based on the Faster RCNN model with a ResNext backbone. The models are trained using Meta’s publicly available [Detectron2](https://ai.meta.com/tools/detectron2/) and [Detectron2go](https://ai.meta.com/blog/d2go-brings-detectron2-to-mobile/) libraries.
* EgoBlur models are only trained to locate the position of faces and license plates of vehicles within color or grayscale images. The models are not used to track or identify individual faces or license plates.
* EgoBlur tooling is available to apply the models to PNG, JPEG and MP4 files.

## Prerequisites

### Installation

Please refer to https://github.com/facebookresearch/EgoBlur/blob/main/README.md#installation.

This installs the EgoBlur Python package, which includes the `egoblur` CLI for images/videos and the `egoblur-vrs-blur` CLI for VRS files.

### Model Downloads

The face and license plate detector models (~400 MB each) must be downloaded separately from the [EgoBlur GitHub repository](https://github.com/facebookresearch/EgoBlur). See the [EgoBlur README](https://github.com/facebookresearch/EgoBlur/blob/main/README.md) for download links and instructions.

### Supported Environments

| OS / Platform                 | OS / Distro Details                                                        | CUDA Toolkit | Supported Python Versions |
| :---------------------------- | :------------------------------------------------------------------------- | :----------- | :------------------------ |
| Linux (x64)                   | Fedora **43**; Ubuntu **20.04 LTS** / **22.04 LTS** / **24.04 LTS**       | 12.8         | 3.10 – 3.12              |
| macOS (Apple Silicon / ARM64) | macOS **>=14 (Sonoma)**                                                   | CPU only     | 3.10 – 3.12            |

:::note
We do not recommend running EgoBlur on macOS. The models run on CPU only on Mac (no CUDA support), which results in very slow inference. For best performance, use a Linux machine with a CUDA-capable GPU.
:::

## Getting Started

EgoBlur provides two tools for applying face and license plate blurring:

### EgoBlur Demo

Python3 tool for blurring faces and license plates in PNG, JPEG, or MP4 files. The tool takes input image/video files and produces blurred output files in the same format.

Go to the [EgoBlur README](https://github.com/facebookresearch/EgoBlur/blob/main/README.md) for detailed usage instructions and examples.

### End-to-end VRS Blur (Gen2)

Python tool for blurring faces and license plates directly in Aria Gen2 VRS files (new in `egoblur` 2.0.1). The `egoblur-vrs-blur` CLI reads camera streams, runs detection, applies blur, and writes everything back into a single output VRS file. Non-camera streams (IMU, VIO, barometer, eye tracking, etc.) are copied verbatim.

```bash
egoblur-vrs-blur \
  --input_vrs_path /path/to/input.vrs \
  --output_vrs_path /path/to/output.vrs \
  --face_model_path /path/to/ego_blur_face_gen2.jit \
  --lp_model_path /path/to/ego_blur_lp_gen2.jit \
  --batch_size 5
```

Run `egoblur-vrs-blur --help` for the full list of options.
