# How to use projectaria_tools type annotation

This page provides information about how to use the python type hinting from stub files (*.pyi).

## How to use type hinting
Please follow the [link](/docs/data_utilities/installation/installation_python) to install projectaria_tools package.

### Type hinting setup for [vscode](https://code.visualstudio.com/download)
* Install python extension
![Image](/img/data_utilities/typehinting/install_python_extension.png)
* Select your own virtual environment on the bottom right corner
![Image](/img/data_utilities/typehinting/select_venv.png)
* Enter the virtual environment path (i.e `$HOME/projectaria_tools_python_env/bin/python`)
![Image](/img/data_utilities/typehinting/enter_venv_path.png)
* Type hinting appears when hover the mouse over the imported functions in your python code
![Image](/img/data_utilities/typehinting/vscode_typehinting_example.png)

### Type hinting setup for [pycharm](https://www.jetbrains.com/pycharm/)
* Add new interpreter
![Image](/img/data_utilities/typehinting/pycharm_select_interpreter.png)
* Enter the virtual environment path (i.e `$HOME/projectaria_tools_python_env/bin/python`)
![Image](/img/data_utilities/typehinting/pycharm_enter_interpreter.png)
* Type hinting appears when hover the mouse over the imported functions in your python code
![Image](/img/data_utilities/typehinting/pycharm_typehinting_example.png)

## Generate and install your own type hinting (advanced users)
* [Download the codebase](/docs/data_utilities/installation/download_codebase)
* Compile the codebase for projectaria_tools python package
```bash
cd $HOME/Documents/projectaria_sandbox/projectaria_tools
pip3 install --upgrade pip
pip3 install --global-option=build_ext --config-settings=compile-args="-j2" .;
```
* Generate python type hinting with generate_stubs.py script
```bash
python3 generate_stubs.py
```
* Install stub files for projectaria_tools package
```bash
pip3 install projectaria_tools-stubs/
```
